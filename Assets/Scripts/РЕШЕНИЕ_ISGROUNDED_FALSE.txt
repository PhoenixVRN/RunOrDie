🔧 РУЧНАЯ НАСТРОЙКА:

ШАГ 1: Проверить GroundCheck
────────────────────────────────────────────────────────────
  1. Выберите Player в Hierarchy
  2. Разверните → увидите GroundCheck внутри
  3. Выберите GroundCheck
  4. В Inspector → Transform → Position:
     • X: 0
     • Y: -0.5  ← ВАЖНО! Должно быть -0.5
     • Z: 0
  
  Если GroundCheck нет:
    - ПКМ на Player → Create Empty
    - Назовите "GroundCheck"
    - Установите Position Y = -0.5


ШАГ 2: Назначить GroundCheck в PlayerController
────────────────────────────────────────────────────────────
  1. Выберите Player
  2. Inspector → Player Controller
  3. Найдите "Ground Check"
  4. Перетащите GroundCheck (из Hierarchy) в это поле


ШАГ 3: Настроить Ground Layer
────────────────────────────────────────────────────────────
  1. Player → Inspector → Player Controller
  2. Найдите "Ground Layer"
  3. В выпадающем списке выберите "Ground"
  
  Если "Ground" нет в списке:
    - Tools → Lode Runner → Настройка проекта
    - Выполните настройку


ШАГ 4: Создать землю под игроком
────────────────────────────────────────────────────────────
  1. GameObject → Lode Runner → Создать землю
  
  ИЛИ вручную:
    - GameObject → 2D Object → Sprite → Square
    - Назовите "Ground"
    - Добавьте BoxCollider2D
    - Inspector → Layer → Ground
    - Расположите ПОД игроком


ШАГ 5: Проверить
────────────────────────────────────────────────────────────
  1. Нажмите Play
  2. Посмотрите в левый верхний угол:
     
     IsGrounded: True ← Должно быть True!
     State: Walking ← Должно быть Walking!

🔍 ДИАГНОСТИКА:

ПРОБЛЕМА 1: GroundCheck слишком высоко
────────────────────────────────────────────────────────────
  СИМПТОМ: IsGrounded всегда False
  
  РЕШЕНИЕ: 
    GroundCheck → Position Y = -0.5
    (должен быть у ног, не в центре персонажа!)


ПРОБЛЕМА 2: Нет земли под игроком
────────────────────────────────────────────────────────────
  СИМПТОМ: Игрок висит в воздухе
  
  РЕШЕНИЕ:
    GameObject → Lode Runner → Создать землю
    Расположите под игроком


ПРОБЛЕМА 3: Земля на неправильном Layer
────────────────────────────────────────────────────────────
  СИМПТОМ: Есть земля, но IsGrounded = False
  
  РЕШЕНИЕ:
    Выберите землю → Inspector → Layer → Ground


ПРОБЛЕМА 4: Ground Layer не выбран в PlayerController
────────────────────────────────────────────────────────────
  СИМПТОМ: GroundCheck не видит землю
  
  РЕШЕНИЕ:
    Player → Player Controller → Ground Layer → Ground


ПРОБЛЕМА 5: Ground Check Radius слишком маленький
────────────────────────────────────────────────────────────
  СИМПТОМ: GroundCheck почти касается земли, но не видит
  
  РЕШЕНИЕ:
    Player Controller → Ground Check Radius = 0.2
    (увеличьте если нужно до 0.3)
