🚀 ЗА 2 МИНУТЫ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ШАГ 1: Откройте префаб Enemy
   Hierarchy → Prefabs → GamePrefabs → Enemies → Enemy

ШАГ 2: Замените компоненты
   ❌ Удалите: EnemyController
   ✅ Добавьте: LodeRunnerEnemyAI
   ✅ Добавьте: GridPathfinder

ШАГ 3: Настройте LodeRunnerEnemyAI
   • Ground Check → GroundCheck (дочерний объект)
   • Ground Layer → Ground + Diggable
   • Ladder Layer → Ladder
   • Rope Layer → Rope
   • Use Simple AI → ✓ (для классического режима)

ШАГ 4: Настройте GridPathfinder
   • Grid Step X → 0.5
   • Grid Step Y → 1.0
   • Ground Layer → Ground + Diggable
   • Ladder Layer → Ladder
   • Draw Debug → ✓ (для визуализации пути)

ШАГ 5: Сохраните префаб
   Ctrl + S

ШАГ 6: Готово! 🎉
   Нажмите Play и проверьте работу

  ✅ Поиск пути через BFS
  ✅ Находит оптимальный маршрут
  ✅ Обходит препятствия
  ✅ Сложнее обмануть

🐛 ВОЗМОЖНЫЕ ПРОБЛЕМЫ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Враг не двигается
   → Проверьте Detection Range (15)
   → Проверьте что игрок в радиусе
   → Проверьте Ground Check назначен

❌ Враг не лазает по лестницам
   → Проверьте Ladder Layer настроен
   → Проверьте что лестницы на слое Ladder

❌ Враг проходит сквозь землю
   → Проверьте Ground Layer = Ground + Diggable
   → Проверьте Rigidbody2D → Gravity Scale = 3
