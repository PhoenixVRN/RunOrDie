❌ Враг не двигается
   ✅ Проверьте Detection Range (15)
   ✅ Проверьте Walk Speed > 0

❌ Враг не лазает по лестницам
   ✅ Проверьте Ladder Layer настроен
   ✅ Проверьте лестницы на слое Ladder

❌ Враг не убивает игрока
   ✅ Проверьте коллайдеры (не Trigger!)
   ✅ Проверьте Rigidbody2D на обоих
