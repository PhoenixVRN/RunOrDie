🐛 ПРОБЛЕМА:

Когда игрок заходил на лестницу сбоку (←→), он начинал падать вниз,
потому что гравитация не отключалась до нажатия ↑/↓.

═══════════════════════════════════════════════════════════════

✅ РЕШЕНИЕ:

Добавлена автоматическая остановка падения при нахождении на лестнице!

Теперь при касании лестницы:
  1. Гравитация автоматически отключается
  2. Вертикальная скорость обнуляется
  3. Игрок не падает вниз
  4. Может свободно двигаться ←→

═══════════════════════════════════════════════════════════════

🎮 КАК ТЕПЕРЬ РАБОТАЕТ:

СЦЕНАРИЙ 1: Заход на лестницу сбоку
────────────────────────────────────────────────────────────
  1. Игрок идет по платформе →
  2. Касается лестницы
  3. ✅ Автоматически останавливается (не падает!)
  4. Может нажать ↑ чтобы лезть вверх
  5. Или продолжить движение →


СЦЕНАРИЙ 2: Переход между лестницами
────────────────────────────────────────────────────────────
  1. Игрок стоит на первой лестнице
  2. Двигается влево ←
  3. Доходит до второй лестницы
  4. ✅ Не падает между лестницами!
  5. Плавно переходит на вторую лестницу


СЦЕНАРИЙ 3: Спрыгивание с лестницы
────────────────────────────────────────────────────────────
  1. Игрок на лестнице
  2. Двигается вбок ← или →
  3. Сходит с лестницы
  4. ✅ Гравитация включается
  5. Начинает падать/идти

🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ:

ЧТО БЫЛО ДОБАВЛЕНО:
────────────────────────────────────────────────────────────
```csharp
// Если просто находимся на лестнице (зашли сбоку) - не падаем!
else if (isOnLadder && currentState != PlayerState.Climbing)
{
    // Отключаем гравитацию, чтобы не падать
    rb.gravityScale = 0f;
    
    // Останавливаем вертикальное движение
    Vector2 vel = rb.linearVelocity;
    vel.y = 0f;
    rb.linearVelocity = vel;
}
```

ЛОГИКА:
────────────────────────────────────────────────────────────
  Условие: isOnLadder = true И currentState != Climbing
  
  Это означает:
    • Игрок касается лестницы (коллайдер пересекается)
    • Но еще не начал лазать (не нажал ↑/↓)
    
  Действие:
    • Гравитация = 0
    • Вертикальная скорость = 0
    • Игрок "замораживается" на месте

💡 ПОВЕДЕНИЕ В РАЗНЫХ СИТУАЦИЯХ:

1. НА ЗЕМЛЕ, ИДЕТ К ЛЕСТНИЦЕ:
   • Гравитация: 3f (нормальная)
   • Падает если нет земли под ногами

2. КАСАЕТСЯ ЛЕСТНИЦЫ СБОКУ:
   • Гравитация: 0f ✅
   • Вертикальная скорость: 0 ✅
   • Не падает!

3. НАЖИМАЕТ ↑ (НАЧИНАЕТ ЛАЗАТЬ):
   • Состояние → Climbing
   • Гравитация: 0f
   • Может лезть вверх/вниз

4. СХОДИТ С ЛЕСТНИЦЫ:
   • Состояние → Walking/Falling
   • Гравитация: 3f
   • Нормальная физика
