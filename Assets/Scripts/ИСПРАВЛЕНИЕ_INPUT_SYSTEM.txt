🚀 ЧТО НУЖНО СДЕЛАТЬ (1 минута):

ШАГ 1: Откройте сцену с игроком

ШАГ 2: Выберите объект "Player" в Hierarchy

ШАГ 3: В Inspector найдите компонент "Player Controller"

ШАГ 4: В самом верху увидите новое поле "Input Actions"

ШАГ 5: Перетащите туда файл "InputSystem_Actions" из папки Assets

ШАГ 6: Нажмите ▶ Play

ШАГ 7: ✓ ВСЁ РАБОТАЕТ!

═══════════════════════════════════════════════════════════════

🎮 УПРАВЛЕНИЕ:

  ← → ↑ ↓  или  WASD  =  Движение
  
  Z  =  Копать яму слева
  
  X  =  Копать яму справа

═══════════════════════════════════════════════════════════════

📸 ГДЕ НАЙТИ InputSystem_Actions:

В панели Project:
  
  Assets/
    └── InputSystem_Actions.inputactions  ← ЭТОТ ФАЙЛ

Просто перетащите его в поле "Input Actions"

═══════════════════════════════════════════════════════════════

⚠️ ЕСЛИ НЕ РАБОТАЕТ:

1. Проверьте Console (Ctrl+Shift+C) на ошибки

2. Убедитесь что Input Actions назначен:
   Player → Player Controller → Input Actions = InputSystem_Actions

3. Перезапустите Unity (File → Save, затем перезапуск)

4. Проверьте Edit → Project Settings → Player:
   Other Settings → Active Input Handling = "Input System Package"
