🎯 ЧТО ТАКОЕ GROUNDCHECK?

GroundCheck - это пустой объект у ног персонажа,
который используется для проверки касания земли.

┌─────────────────────┐
│   🧍 Player          │
│   ├─ SpriteRenderer │
│   ├─ Collider       │
│   └─ 📍 GroundCheck  │  ← Находится у ног (Y=-0.5)
│                     │
└─────────────────────┘
        │
────────┴──────────── Земля

Когда GroundCheck касается земли → isGrounded = true
Это нужно для:
  • Определения, когда можно копать
  • Проверки состояния (стоит/падает)
  • Корректной работы физики
