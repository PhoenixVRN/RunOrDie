💡 СОВЕТЫ ПО ДИЗАЙНУ УРОВНЕЙ:
═══════════════════════════════════════════════════════════════

✅ ХОРОШИЕ ПРАКТИКИ:

• Давайте игроку время привыкнуть к механике
  └→ Первые уровни - простые переходы

• Комбинируйте веревки с другими механиками
  └→ Лестницы, копание, враги

• Создавайте выбор: быстрый но опасный VS медленный но безопасный
  └→ Веревка над ямой VS обход по земле

• Визуально выделяйте веревки
  └→ Яркий цвет, анимация покачивания

• Тестируйте разные последовательности
  └→ Можно ли пройти несколькими путями?


❌ ЧАСТЫЕ ОШИБКИ:

• Слишком узкие зоны захвата
  └→ Игрок промахивается, это фрустрирует

• Слишком много веревок подряд
  └→ Становится скучно и монотонно

• Нет альтернативных путей
  └→ Застрял = выход из игры

• Веревки на одинаковой высоте без вариаций
  └→ Нет челленджа

• Нет визуальных подсказок
  └→ Игрок не понимает, что нужно делать

  ✓ Отметьте старт и финиш
  ✓ Определите путь игрока

ШАГ 2: Размещение платформ
  ✓ Создайте основные платформы (Ground)
  ✓ Оставьте промежутки для ям

ШАГ 3: Добавление веревок
  ✓ Перетащите префаб Rope в сцену
  ✓ Расположите на нужной высоте
  ✓ Проверьте размер (BoxCollider2D)

ШАГ 4: Настройка
  ✓ Убедитесь что Layer = Rope
  ✓ Is Trigger = ✓
  ✓ Проверьте зону захвата (Gizmos)

ШАГ 5: Тестирование
  ✓ Запустите игру
  ✓ Проверьте захват веревки
  ✓ Отрегулируйте позицию при необходимости

ШАГ 6: Полировка
