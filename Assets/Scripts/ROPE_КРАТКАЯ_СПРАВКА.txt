⚡ БЫСТРАЯ НАСТРОЙКА (3 КЛИКА):

🐛 РЕШЕНИЕ ПРОБЛЕМ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Не хватается
   → Проверь Rope Layer в PlayerController
   → Проверь что префаб веревки имеет слой "Rope"
   → Веревка должна быть на уровне рук

❌ Не двигается
   → Rope Speed должен быть > 0

❌ Застревает
   → Нажми ↓ чтобы отпустить

• Игрок автоматически центрируется по высоте
• Можно комбинировать с лестницами
• centerOnRope = false для свободного висения
