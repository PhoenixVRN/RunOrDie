🐛 ПРОБЛЕМА:

При старте игры персонаж не двигается при нажатии кнопок.

═══════════════════════════════════════════════════════════════

🔍 ОТЛАДОЧНАЯ ИНФОРМАЦИЯ НА ЭКРАНЕ:

При запуске игры в левом верхнем углу появится информация:

  State: Walking/Falling/Climbing
  IsGrounded: True/False
  Input: (x, y)
  Velocity: (x, y)

Это поможет найти проблему!

═══════════════════════════════════════════════════════════════

✅ БЫСТРОЕ ИСПРАВЛЕНИЕ:

РЕШЕНИЕ 1: Использовать автоисправление
────────────────────────────────────────────────────────────
  Unity → Tools → Lode Runner → 🔧 Быстрое исправление Player
  
  Это автоматически:
    ✓ Создаст и назначит GroundCheck
    ✓ Назначит InputActions
    ✓ Исправит все настройки


РЕШЕНИЕ 2: Создать нового игрока
────────────────────────────────────────────────────────────
  GameObject → Lode Runner → Создать игрока
  
  Всё настроится автоматически!

═══════════════════════════════════════════════════════════════

🔍 ДИАГНОСТИКА ПО СИМПТОМАМ:

СИМПТОМ 1: State = Falling, IsGrounded = False
────────────────────────────────────────────────────────────
  ПРОБЛЕМА: GroundCheck находится не там где нужно
  
  РЕШЕНИЕ:
    1. Выберите Player → GroundCheck
    2. Установите Position Y = -0.5
    3. Проверьте, что GroundCheck у ног персонажа


СИМПТОМ 2: Input = (0.00, 0.00) при нажатии кнопок
────────────────────────────────────────────────────────────
  ПРОБЛЕМА: InputActions не назначен
  
  РЕШЕНИЕ:
    1. Выберите Player в Hierarchy
    2. Inspector → Player Controller → Input Actions
    3. Перетащите InputSystem_Actions из Assets/


СИМПТОМ 3: State = Walking, но Velocity = (0, 0)
────────────────────────────────────────────────────────────
  ПРОБЛЕМА: Возможно проблема с физикой
  
  РЕШЕНИЕ:
    1. Проверьте Rigidbody2D:
       - Body Type = Dynamic
       - Simulated = ✓
       - Constraints = Freeze Rotation Z
    2. Проверьте коллайдер персонажа


СИМПТОМ 4: Ошибки в Console
────────────────────────────────────────────────────────────
  Откройте Console (Ctrl+Shift+C) и прочитайте ошибки:
  
  "GroundCheck не назначен"
    → Используйте Tools → Lode Runner → Быстрое исправление
  
  "InputActions не назначен"
    → Назначьте InputSystem_Actions в Inspector

═══════════════════════════════════════════════════════════════

📋 ЧЕКЛИСТ ПРОВЕРКИ:

□ GroundCheck создан и находится у ног (Y = -0.5)
□ GroundCheck назначен в Player Controller
□ InputSystem_Actions назначен в Player Controller
□ Ground Layer выбран в Player Controller
□ Под игроком есть земля с Layer = Ground
□ Rigidbody2D: Body Type = Dynamic
□ Rigidbody2D: Simulated = ✓
□ В Console нет ошибок

═══════════════════════════════════════════════════════════════

🔧 ПРОВЕРКА НАСТРОЕК:

1. GROUNDCHECK:
────────────────────────────────────────────────────────────
   Player
   └─ GroundCheck  ← Должен быть здесь!
      Position: (0, -0.5, 0)  ← Важно!


2. PLAYER CONTROLLER:
────────────────────────────────────────────────────────────
   ☑ Input Actions: InputSystem_Actions
   ☑ Ground Check: GroundCheck (Transform)
   ☑ Ground Check Radius: 0.2
   ☑ Ground Layer: Ground


3. RIGIDBODY2D:
────────────────────────────────────────────────────────────
   Body Type: Dynamic
   Material: None
   Simulated: ✓
   Use Auto Mass: ✓
   Mass: 1
   Linear Drag: 0
   Angular Drag: 0.05
   Gravity Scale: 3
   Constraints: Freeze Rotation Z ✓


4. ЗЕМЛЯ ПОД ИГРОКОМ:
────────────────────────────────────────────────────────────
   Должна быть платформа/земля:
     • С коллайдером (BoxCollider2D)
     • Layer = Ground
     • Под игроком!

═══════════════════════════════════════════════════════════════

🎮 ТЕСТИРОВАНИЕ:

ТЕСТ 1: Визуальная проверка
────────────────────────────────────────────────────────────
  1. Запустите игру
  2. Посмотрите в левый верхний угол
  3. Проверьте информацию:
  
     State: Walking ← Должно быть это!
     IsGrounded: True ← И это!
     Input: (1.00, 0.00) ← При нажатии →
     Velocity: (5.00, 0.00) ← Должна быть скорость!


ТЕСТ 2: Ручная проверка
────────────────────────────────────────────────────────────
  1. Выберите Player в Hierarchy (во время игры)
  2. Inspector → Player Controller
  3. Проверьте переменные в разделе Debug


ТЕСТ 3: Scene View
────────────────────────────────────────────────────────────
  1. Во время игры переключитесь в Scene
  2. Выберите Player
  3. Должны быть видны Gizmos:
     • Зеленый круг у ног = GroundCheck
     • Желтые круги = зоны копания

═══════════════════════════════════════════════════════════════

💡 ЧАСТЫЕ ОШИБКИ:

1. GroundCheck не у ног
   → Установите Position Y = -0.5

2. Ground Layer не выбран
   → Inspector → Ground Layer → выберите Ground

3. Нет земли под игроком
   → Создайте платформу с Layer = Ground

4. InputActions не назначен
   → Перетащите InputSystem_Actions

5. Rigidbody2D: Simulated выключен
   → Включите галочку Simulated

6. Player на неправильном Layer
   → Player должен быть на Default layer
