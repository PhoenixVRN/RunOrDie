🎯 В ИГРЕ ДВА ВИДА ЗЕМЛИ:

1. 🟩 GROUND - обычная земля
   • Нельзя копать
   • Постоянная
   • Layer: Ground

2. 🟫 DIGGABLE - копаемая земля
   • Можно копать
   • Восстанавливается
   • Layer: Diggable

✅ РЕШЕНИЕ: НАСТРОИТЬ GROUND LAYER

PlayerController должен видеть ОБА слоя как землю!

═══════════════════════════════════════════════════════════════

🔧 КАК НАСТРОИТЬ (30 секунд):

ШАГ 1: Выберите Player в Hierarchy

ШАГ 2: Inspector → Player Controller → Ground Layer

ШАГ 3: Кликните на выпадающий список "Ground Layer"

ШАГ 4: Выберите ОБА слоя:
  ☑ Ground     ← Поставьте галочку
  ☑ Diggable   ← Поставьте галочку

ШАГ 5: Нажмите Play

ШАГ 6: ✓ Готово! Теперь IsGrounded = True на любой земле!

💡 ПОЧЕМУ ТАК НУЖНО?

GroundCheck проверяет касание земли с помощью LayerMask.

Если выбран только Ground:
  ✓ Видит обычную землю
  ❌ НЕ видит копаемую землю
  → IsGrounded = False на копаемых блоках!

Если выбраны Ground И Diggable:
  ✓ Видит обычную землю
  ✓ Видит копаемую землю
  → IsGrounded = True на любой земле! ✓

🎮 ГЕЙМПЛЕЙНЫЕ ПОСЛЕДСТВИЯ:

ПРАВИЛЬНАЯ НАСТРОЙКА (Ground + Diggable):
────────────────────────────────────────────────────────────
  ✅ Можно ходить по обычной земле
  ✅ Можно ходить по копаемым блокам
  ✅ Можно копать с копаемых блоков
  ✅ IsGrounded работает везде


НЕПРАВИЛЬНАЯ НАСТРОЙКА (только Ground):
────────────────────────────────────────────────────────────
  ✅ Можно ходить по обычной земле
  ❌ НЕ видит копаемые блоки как землю
  ❌ Падает сквозь копаемые блоки
  ❌ Не может копать

ЗАЧЕМ ДВА РАЗНЫХ ПОЛЯ?

  • Ground Layer - для проверки земли (GroundCheck)
    → Нужны ОБА: Ground + Diggable
  
  • Diggable Layer - для поиска копаемых блоков
    → Нужен только Diggable

═══════════════════════════════════════════════════════════════

🔍 ПРОВЕРКА ПРАВИЛЬНОСТИ:

ТЕСТ 1: Обычная земля
────────────────────────────────────────────────────────────
  1. Создайте землю (Layer = Ground)
  2. Встаньте на неё
  3. IsGrounded: True ← Должно быть!


ТЕСТ 2: Копаемая земля
────────────────────────────────────────────────────────────
  1. Создайте копаемый блок (Layer = Diggable)
  2. Встаньте на него
  3. IsGrounded: True ← Должно быть!


ТЕСТ 3: Копание
────────────────────────────────────────────────────────────
  1. Встаньте на копаемый блок
  2. Создайте копаемый блок рядом
  3. Нажмите Z или X
  4. Блок должен выкопаться ← Работает!
