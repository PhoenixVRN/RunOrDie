📦 ЧТО ВКЛЮЧЕНО:
  • LevelPropSnap.cs - основной компонент
  • LevelPropSnapEditor.cs - кастомный инспектор с кнопками
  • LevelPropSnapMenuItem.cs - меню и горячие клавиши

═══════════════════════════════════════════════════════════════

🎮 КАК ИСПОЛЬЗОВАТЬ:

1. ДОБАВИТЬ КОМПОНЕНТ К ОБЪЕКТУ:
   • Выберите пропс в сцене
   • Добавьте компонент "Level Prop Snap"
   • Или: Tools → Level Props → Добавить LevelPropSnap к выбранным

2. НАСТРОЙКИ В ИНСПЕКТОРЕ:
   ┌─────────────────────────────────────────┐
   │ Grid Step X: 0.5  (по умолчанию)       │
   │ Grid Step Y: 1.0  (по умолчанию)       │
   │ Grid Step Z: 0.5  (по умолчанию)       │
   │                                         │
   │ ☑ Snap X - выравнивать по оси X        │
   │ ☑ Snap Y - выравнивать по оси Y        │
   │ ☐ Snap Z - выравнивать по оси Z        │
   └─────────────────────────────────────────┘

3. КНОПКА ВЫРАВНИВАНИЯ:
   В инспекторе объекта появится зелёная кнопка:
   
   ┌───────────────────────────────────────┐
   │  ⚡ ВЫРОВНЯТЬ ПО СЕТКЕ ⚡              │
   └───────────────────────────────────────┘
   
   При нажатии объект выровняется:
   • X к значениям 0.5, 1.5, 2.5, 3.5 и т.д.
   • Y к значениям 0.5, 1.5, 2.5, 3.5 и т.д.
   • Z кратно 0.5 (если включено)

4. ДОПОЛНИТЕЛЬНЫЕ КНОПКИ:
   ┌────────────┬────────────┬──────────┐
   │ X: ...0.5  │ Y: ...0.5  │ Z: 0.5   │
   └────────────┴────────────┴──────────┘
   
   Выравнивают только по выбранной оси

═══════════════════════════════════════════════════════════════

⌨️ ГОРЯЧИЕ КЛАВИШИ:

  Ctrl + Shift + S - выровнять все выбранные объекты

═══════════════════════════════════════════════════════════════

📋 МЕНЮ TOOLS:

  Tools → Level Props →
    • Выровнять выбранные объекты (Ctrl+Shift+S)
    • Добавить LevelPropSnap к выбранным
    • Выровнять все пропсы на сцене

═══════════════════════════════════════════════════════════════

🎯 БЫСТРЫЙ ДОСТУП:

  Правый клик на объект в иерархии →
  GameObject → Level Props → Выровнять по сетке

═══════════════════════════════════════════════════════════════

💡 СОВЕТЫ:

  1. Добавьте компонент на префабы пропсов - тогда все
     новые экземпляры будут иметь возможность выравнивания

  2. Используйте Ctrl+Shift+S для массового выравнивания
     нескольких объектов одновременно

  3. Настройте шаг сетки под ваши нужды:
     • Для пиксель-арта: 0.0625, 0.125, 0.25
     • Для крупных объектов: 1.0, 2.0, 5.0

  4. В режиме Scene выбранный объект с компонентом
     показывает зелёный крестик на своей позиции

  5. Все действия поддерживают Undo (Ctrl+Z)

═══════════════════════════════════════════════════════════════

📊 ПРИМЕР ИСПОЛЬЗОВАНИЯ:

  1. Создайте пропс (куб, платформа, препятствие)
  2. Добавьте компонент LevelPropSnap
  3. Расставьте несколько копий объекта примерно
  4. Выберите все копии
  5. Нажмите Ctrl+Shift+S
  6. ✓ Все объекты идеально выровнены по сетке!
